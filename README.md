## 👓 src/js 구조 요약
1. ./app 
- Todo Application의 모든 기능을 담당
2. ./component
- Todo Application의 모든 UIComponent들의 초기화 및 이벤트 바인딩 담당
3. ./datastore
- Todo Application의 데이터 엑세스 담당
4. ./util
- Todo Application에서 공통적으로 쓰이는 함수모듈
5. ./index.js
- Todo Application의 엔트리 포인트
